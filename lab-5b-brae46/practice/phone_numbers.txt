GUIDELINES:
012-345-6789
012_________: area code
____345_____: exchange code
________6789: line number

* Can have 7 or 10 digits
* Dashes are optional
* Phone numbers with dashes must follow 000-000-0000 or 000-0000 pattern
* No special characters or letters
* A number with 555 exchange code is not valid (ignore area codes)
